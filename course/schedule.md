# CS 7180: 15-Week Schedule

## Overview

| Weeks | Focus |
|-------|-------|
| 1-3 | Foundations: LLM fundamentals, Mom Test, prompt engineering |
| 4-6 | Cursor IDE, TDD intro, CI/CD basics, **Project 1 due** |
| 7-10 | Advanced prompting, context engineering, evals, **No-AI Challenge** |
| 11-13 | Agile, parallel agents, advanced CI/CD, **Project 2 due** |
| 14-15 | Production readiness, monitoring, **Project 3 due** |

## Detailed Weekly Schedule

| Week | Module | Topics | Deliverables | Project Status |

| **6** | **Project 1 + Modality 3** | **PROJECT 1 PRESENTATIONS**, Claude Code & Terminal AI, Agentic coding intro, Terminal workflows, Automation & scripts | **PROJECT 1 DUE**, Weekly Quiz 6 | **P1:** Final submission, **P2:** Planning begins |
| **7** | **Advanced Prompting** | Chain-of-thought, Meta-prompting, Prompt chaining, Structured prompts (XML/JSON), Cost optimization, Building prompt libraries | **HW3 DUE:** Context, Weekly Quiz 7 | **P2:** Sprint 1, Architecture & setup |

| **10** | **TDD + CI/CD + Evals** | Advanced TDD with AI, Multi-stage CI/CD pipelines, Deploy previews, Coverage reporting, Security scanning, **Evals Part 2:** LLM-as-judge, Automated dashboards | **HW4 DUE:** TDD+CI/CD, **MIDTERM:** No-AI Challenge, Weekly Quiz 10 | **P2:** Final polish, Complete eval suite |
| **11** | **Project 2 + Agile** | **PROJECT 2 PRESENTATIONS**, Agile workflows with AI, Scrum ceremonies, Sprint planning with AI, Team velocity & estimation, **Team Formation Workshop** | **PROJECT 2 DUE**, Weekly Quiz 11 | **P2:** Final submission, **P3:** Team formation |
| **12** | **Parallel Agents** | Agent fundamentals: ReAct, Plan-and-Execute, Agent architectures, Single agent workflows, Multi-agent coordination, Agent safety & debugging, Cost control for agents | **HW5 DUE:** Parallel agents, Weekly Quiz 12 | **P3:** Sprint 1, Parallel development |

| **14** | **Production** | Monitoring & observability, Error tracking & logging, Performance optimization, Security best practices, Accessibility (WCAG basics), Professional documentation, Portfolio building tips | Weekly Quiz 14 | **P3:** Sprint 3, Polish & deploy |
| **15** | **Demo Day** | **FINAL PRESENTATIONS**, Portfolio showcase, Course retrospective, Industry panel (optional), Career prep discussion | **PROJECT 3 DUE** | **P3:** Final demos, Portfolio complete |
